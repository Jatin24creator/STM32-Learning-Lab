# STM32C031C6 Board

This folder contains all experiments performed using the STM32C031C6 microcontroller.

## MCU Overview
- ARM Cortex-M0+
- Entry-level STM32 C0 series
- Designed for cost-sensitive and low-power applications

## Toolchain

- STM32 HAL (CubeMX-generated)
